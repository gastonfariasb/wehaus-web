WeHaus




Home Automation made simple

Features


Remote Control

Brings a built-in infrared LED which enables you to control any device within reach.

Motion Sensor

Detects movement in front of the WeHaus, feeding this information to you, so you can react acordingly

LED Notifications

With its RGB LED you can configure it to let you know about the state of the virtual world.

Power Switches

One of its most powerful features. Conect any device to this power source and start controling it remotely. Popular choices include lights.

Plug n' Play

Virtualy no setup process. Instant gratification

Extensible Network

Acts as a hub, so you can connect all kinds of things
